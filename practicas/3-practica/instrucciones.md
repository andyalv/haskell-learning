# Práctica 3
## Primer función
Realizar una función que deuelva el promedio y comportamiento de una lista de números

**Comportamiento de la lista:**
- Promedio menor a $100$ = "BAJO"
- Promedio menor a $500$ = "NORMAL"
- Promedio mayor o igual a $500$ = "ALTO"

**Consideraciones:**
- Generar la función con tipo explícito
- Usar los tipos de datos y clases correctos
- Usar ajuste de patrones, guardas, `otherwise`, `where` o `case`
- Realizar todas las validaciones para todos los posibles casos (ajuste de patrones)

> [!note]  **Nota**
> Considerar usar la función `div` en lugar de `/`

## Segunda función
Realizar una función que permita obtener el voltaje dadas la corriente y la resistencia ($V=I*R$) e indicar como es el voltaje.

**Condiciones de voltaje:**
- Mayor a $60V$ = bajo
- Mayor a $120V$ = normal
- Mayor a $220V$ = alto

**Consideraciones:**
- Realizar todas las validaciones para todos los posibles casos
- Las condiciones de voltaje deben ir almacenadas en variables tanto para la función usando guardas como para la función usando la cláusula `where`
- Usar los tipos de datos y clases correctos
